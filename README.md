# Airline Engineering Control Center Dashboard

A full-stack airline fleet monitoring dashboard built with Flask.

## Features
- Fleet reliability monitoring
- MTBF analytics
- Defect tracking system
- Predictive maintenance model
- Interactive charts

## Tech Stack
- Python
- Flask
- SQLAlchemy
- Chart.js
- HTML/CSS

## System Modules
- Fleet reliability service
- Predictive maintenance model
- Defect reporting system
- Aircraft health dashboard

## Run Locally

```bash
pip install -r requirements.txt
python app.py
